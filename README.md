Programming Coursework
======================

This repository contains selected programming assignments completed as part of
various courses in programming and data analysis.

The purpose of this repository is to showcase academic work and practical
experience using different programming languages, including:

- Python
- R
- SQL 

Each assignment is organized by course.

Repository Structure
--------------------
The repository is organized as follows:

- Each top-level folder corresponds to a programming language 
- Subfolders contain individual courses and assignments
- Code files, reports, and supporting materials are stored together for each
  assignment

Notes
-----
- This code was written for educational purposes.
